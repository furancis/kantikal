# Shape Lock

Do not lose this shape:

Suno Visual Studio is a dynamic visual app for the whole Suno music generation workflow. It is not a CLI-first product and not a ComfyUI-only music-video tool. The core flow is brief -> prompt/lyrics/style/voice -> Suno batches -> compare/rate/version -> Song Lab edits/stems -> optional Music Video Lane -> release pack.

The Music Video Lane is subordinate. It opens from a selected song/version, keeps audio as source of truth, builds storyboard and ComfyUI render plans, then blocks export until perfect lipsync QA passes.
